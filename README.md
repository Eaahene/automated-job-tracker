# Job Aggregator Pipeline

A Python-based ETL pipeline that fetches jobs from Ashby job boards and stores them in:

- CSV files
- Google Sheets

## Features
- API Integration
- Automated scheduling
- Cloud synchronization
- Data persistence
- Error handling

## Technologies
- Python
- Requests
- Pandas
- Google Sheets API
- gspread

## Setup

1. Clone repo
2. Install requirements
3. Add service-account.json
4. Run script
